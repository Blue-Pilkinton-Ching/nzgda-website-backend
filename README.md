## Stack

- Express
- Typescript
- Firebase
- AWS for Cloud storage

## Getting Started

Install dependencies:

```bash
npm install
```

Next, run the development server:

```bash
npm run dev
```

Server routes are hosted on [http://localhost:3001](http://localhost:3001)
